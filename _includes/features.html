Core features

  • Per-Condition Detectors
    Fine-grained detectors for each movement / click condition to reduce false positives.
  • Runtime Training
    Models adapt over time using safe, server-side persisted data.
  • Packet-level Sanitation
    Integrates with PacketEvents to sanitize inbound packets and block exploit flows.

Operational advantages

Minimal CPU footprint, asynchronous subsystems, clear operator observability and logs.
